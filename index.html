vue-tableau-slider

A simple slider same as tableau.

Get Started →

Three slider modes

Currently supports value range, at least, at most three modes.

Support value out of range

Support the smaller value is less than the minimum value, and the larger value is greater than the maximum value.

Support css scale zoom

Compatible with outer element scale scaling and slider drag events.
